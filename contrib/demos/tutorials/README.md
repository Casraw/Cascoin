# Tutorials

Schritt-für-Schritt Anleitungen für Cascoin L2 und Smart Contracts.

## Verfügbare Tutorials

| Tutorial | Beschreibung | Schwierigkeit |
|----------|--------------|---------------|
| [01_l2_quickstart.md](01_l2_quickstart.md) | L2 Demo Chain Setup und Grundlagen | Anfänger |
| [02_token_creation.md](02_token_creation.md) | ERC-20 Token mit Trust-Integration erstellen | Fortgeschritten |

## Voraussetzungen

Bevor Sie mit den Tutorials beginnen:

1. **Cascoin Core gebaut** - Siehe [Build-Anleitung](../../../doc/build-linux.md)
2. **Bash-Shell** - Version 4.0 oder höher
3. **Grundkenntnisse** - Blockchain-Konzepte und CLI-Nutzung

## Empfohlene Reihenfolge

1. Starten Sie mit dem **L2 Quick Start** um die Demo-Umgebung einzurichten
2. Fahren Sie mit **Token Creation** fort um Smart Contracts zu verstehen
